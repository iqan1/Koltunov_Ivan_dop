# C++ Advanced Tasks

Колтунов Иван 4341 Дополнительные задания ООП.

## Содержание

### [Task 1: SimpleRNG](./Task1_SimpleRNG)
Реализация генератора псевдослучайных чисел с поддержкой `InputIterator`.
- Используется метод остатков.
- Реализована защита от переполнения памяти при бесконечном цикле.

### [Task 2: Mask](./Task2_Mask)
Шаблонный класс `Mask<N>` для операций над контейнерами.
- Методы `slice`, `transform`, `slice_and_transform`.
- Compile-time проверки размера маски.

### [Task 3: MemReserver](./Task3_MemReserver)
Менеджер статической памяти без использования STL.
- Реализован через `placement new`.
- Поддерживает создание и удаление объектов, контроль ошибок.

### [Task 5: Pipeline](./Task5_Pipeline)
Система пайплайнов для обработки данных.
- Синтаксис через оператор `|`.
- Поддержка ленивых вычислений.
